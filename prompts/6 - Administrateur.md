Admin Dashboard – Full Feature List

1. User Management

View all registered users across all 7 role types

Filter by status: All / Pending / Approved / Rejected

Approve or reject accounts with real-time UI update

User cards showing name, email, role, and registration date

2. Pending Approvals Panel

Dedicated section highlighting accounts waiting for review

Quick approve / reject actions without opening full profile

Badge counter showing total pending accounts

3. User Detail View

Full profile of any registered user

View submitted information per role (e.g. doctor license, pharmacy address)

Approve, reject, or suspend the account from this view

4. Role-Based User Stats

Count of users per role type

Count of pending / approved / rejected per role

5. Account Suspension

Ability to suspend or deactivate any active account after approval
