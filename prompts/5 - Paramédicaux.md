Paramedical Dashboard – Features to Develop

Access: role: paramedical · Base route: /paramedical-dashboard

1. Overview · /paramedical-dashboard

KPI cards: followed patients, today's consultations, weekly hours, monthly revenue

11-tile quick-action grid

Recent activity feed

2. Appointments · /paramedical-dashboard/appointments

Appointment list by patient, care type, location (home / clinic), and status

Add and modify appointment actions

3. Planning · /paramedical-dashboard/planning

Daily and weekly view with chronological visit order

Status per visit: done / in progress / to-do

Daily mileage and time estimate

4. Patients · /paramedical-dashboard/patients

Searchable patient card grid

Shows condition, status, and next appointment per patient

Open medical file or message patient actions


****************************************************************



5. Care Record · /paramedical-dashboard/care-record

Care session form: patient, care type, clinical notes, photo upload, digital signature

Submit to validate the session

6. Vitals · /paramedical-dashboard/vitals

Per-patient vitals entry form: BP, heart rate, temperature, SpO2, glucose

Inline alerts for abnormal values

7. Teleconsultation · /paramedical-dashboard/teleconsultation

Available doctors list

Initiate video or audio call

Active call screen with timer, image sharing, and chat

8. Messaging · /paramedical-dashboard/messaging

Two-panel chat with doctors and administration

Unread count badges

Send message on Enter


****************************************************************


9. Notifications · /paramedical-dashboard/notifications

Typed alerts: urgent vitals, doctor messages, schedule changes, prescriptions

Bulk mark-as-read action

Act button on urgent items

10. Map · /paramedical-dashboard/map

Optimized visit route with numbered patient markers on map

Visit order list with address and distance

Total km calculation

11. Supplies · /paramedical-dashboard/supplies

Medical supplies inventory with stock level bars

Low-stock banner alert

Delivery request button for low-stock items

12. Reports · /paramedical-dashboard/reports

Daily and weekly care summary reports with visit count and hours

Downloadable PDF

Care history log

13. Settings · /paramedical-dashboard/settings

Edit profile: name, phone, specialization

Notification toggles: email, SMS, push, daily report
