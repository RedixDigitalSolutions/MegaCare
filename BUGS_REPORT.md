### Top Priorities

1. **Fix broken lucide-react imports** (bugs #1–5) — these cause TypeScript build failures or silent render crashes.
2. **Implement real authentication** (bugs #6–7) — currently any credentials are accepted, no backend is called.
3. **Implement backend routes** (bugs #13–18) — all 6 route files return 501.
4. **Fix non-functional UI features** (bugs #19–22) — prescription form, add-to-cart, and doctor profile use real URLs/buttons that silently do nothing.
5. **Remove all `console.log` statements** from production code (bugs #25–34) — some log sensitive user data.
