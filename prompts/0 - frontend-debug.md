#Frontend – Debug & Error Fix

Perform a full audit of the frontend codebase and fix all identified issues.

* Task Details:

- Check the browser console for any errors or warnings and resolve them
- Fix any broken routes or pages that fail to load
- Resolve any component crashes or rendering issues
- Fix any broken imports, missing dependencies, or undefined variables
- Check and fix any layout or styling breakages across all pages
- Ensure all API calls are handled correctly with proper error states
- Verify all role-based routing and access guards work as expected
- Test all pages at both desktop and mobile and fix any responsive issues
